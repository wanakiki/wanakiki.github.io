合宙 Air780E

IoT 通信模组

基于紫光展锐 EC618 平台的 LTE Cat.1 无线通信模组，支持全网通。内置 LuatOS 框架，支持 Lua 脚本二次开发，是 2G 替换方案的理想选择。

  • 紫光展锐 EC618 芯片，主频 533MHz
  • LTE Cat.1 全网通，下行 10Mbps / 上行 5Mbps
  • 16MB Flash + 16MB RAM，支持 Lua 脚本开发
  • 丰富外设：USB 2.0、SPI、I2C、UART、ADC、GPIO
  • 支持 LuatOS 框架与 FOTA 远程升级
📖 官方文档 🏢 合宙官网

合宙 Air700E

IoT 通信模组

合宙推出的超小尺寸 LTE Cat.1 通信模组，主打极致性价比和小体积。基于 EC618 平台，适合对尺寸和成本有严格要求的物联网终端产品。

  • 基于紫光展锐 EC618 平台，LTE Cat.1 全网通
  • 超小封装（17.7mm × 15.8mm），适合紧凑设计
  • 16MB Flash + 16MB RAM，支持 LuatOS 开发
  • 支持 FOTA 远程升级
  • 高性价比，适合大规模物联网部署
📖 官方文档 🏢 合宙官网

合宙 Air001

MCU 开发板

合宙推出的超低成本微控制器开发板，基于 ARM Cortex-M0+ 内核，主打极致性价比，适用于替代传统 8 位单片机的场景。

  • ARM Cortex-M0+ 内核，主频 72MHz
  • 32KB Flash + 4KB SRAM
  • 极致低价，适合大批量低成本应用
  • 支持 Keil MDK、IAR、GCC 开发工具链
  • 外设：ADC、UART、SPI、I2C、TIM、GPIO
📖 官方文档 🏢 合宙官网

Raspberry Pi Pico

MCU 开发板

树莓派基金会推出的基于 RP2040 芯片的低成本微控制器开发板。双核 Cortex-M0+ 架构，具备独特的 PIO 可编程 I/O 系统，支持 MicroPython 和 C/C++。

  • RP2040 芯片：双核 ARM Cortex-M0+，133MHz
  • 264KB SRAM + 2MB 板载 Flash
  • 26 个多功能 GPIO，支持 SPI/I2C/UART/PWM/ADC
  • 独特的 PIO 子系统，支持自定义通信协议
  • 支持 MicroPython / C/C++ SDK，UF2 烧录
📖 官方文档 🏢 树莓派官网

Seeed Wio Terminal

MCU 开发板

基于 SAMD51 处理器的集成开发平台，配备 2.4 英寸 LCD 屏幕、Wi-Fi/蓝牙模块及多种传感器。兼容 Arduino 和 MicroPython，适合快速原型开发。

  • ATSAMD51P19 MCU：ARM Cortex-M4，120MHz，4MB Flash
  • 内置 2.4 英寸 320×240 TFT LCD 显示屏
  • 集成 Realtek RTL8720DN 双频 Wi-Fi + 蓝牙 5.0
  • 五向摇杆、侧边按键、加速度计、麦克风、红外
  • 兼容 40-pin Raspberry Pi GPIO 和 Grove 接口
📖 官方文档 🏢 Seeed 官网

Seeed XIAO ESP32-C3

MCU + Wi-Fi/BLE

XIAO 系列中基于乐鑫 ESP32-C3 的超紧凑开发板，RISC-V 32 位单核处理器，集成 Wi-Fi 和蓝牙 5.0，尺寸仅拇指大小。

  • 乐鑫 ESP32-C3：RISC-V 32 位单核，160MHz
  • 400KB SRAM + 4MB Flash
  • 2.4GHz Wi-Fi (802.11 b/g/n) + 蓝牙 5.0 BLE
  • 超小尺寸 21mm × 17.5mm
  • 11 个数字 I/O、4 个模拟 I/O，USB-C 供电
📖 官方文档 🏢 Seeed 官网

Microchip AVR64DD32 Curiosity Nano

MCU 开发板

Microchip Curiosity Nano 系列开发板，搭载 AVR64DD32 MCU。支持多电压 I/O（MVIO），内置调试器，开箱即用。

十二指神探

多功能调试助手

硬禾学堂推出的基于 RP2040 的多功能硬件调试助手。通过 12 根引脚（9 根 IO + 3.3V/5V/GND）提供灵活的调试功能，使用 MicroPython 编程，可实现逻辑分析仪、信号发生器、协议调试等多种用途。

  • 基于 RP2040 双核 Cortex-M0+，主频 133MHz（可超频至 250MHz）
  • 2MB Flash，12MHz 晶体，板载 5V→3.3V 降压（800mA）
  • 12 引脚设计：9 根 IO（GPIO20-28）+ 3.3V + 5V + GND
  • 支持逻辑分析仪（PulseView）、PWM、SPI/I2C/UART、ADC 采集
  • 可实现 DDS 波形发生、DAPLink 调试器、VGA/HDMI 信号生成等
  • Type-C USB 供电/下载/通信，支持 MicroPython 和 C/C++
